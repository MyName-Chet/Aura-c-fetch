# My Project

## Description
This project is designed to provide a simple way to retrieve and display system information along with an image using the chafa tool. The main functionality is encapsulated in a Bash script that formats the output for better readability.

## Folder Structure
```
my-project
├── assets
│   └── logo.jpg
├── src
│   ├── app.ts
│   └── types
│       └── index.ts
├── install.sh
└── README.md
```

## Prerequisites
Before running this project, ensure that you have the chafa tool installed on your Ubuntu system. You can install it using the following command:

```bash
sudo apt update && sudo apt install chafa -y
```

## Installation
To set up the project, run the `install.sh` script. This will guide you through the installation process.

## Usage
To use the main functionality of the project, execute the `aura.sh` script located in the `src` directory. This will display the system information along with the logo image.
